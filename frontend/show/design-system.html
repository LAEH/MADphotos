Token Architecture

Three-layer token architecture with three theme scopes:

  1. Primitive tokens — Apple System Colors, raw values (e.g. --system-blue, --system-gray-3)
  2. Semantic tokens — Purpose-based (e.g. --text, --bg-elevated, --glass-border)
  3. Component tokens — Scoped to specific UI patterns (e.g. --c-vibe, --emo-happy)

Intentionally Not Tokenized

Body font-size (14px base definition) · Display sizes (19px, 9px between token steps) · Icon/emoji font-sizes (20-24px on buttons) · Component dimensions (44px buttons, 48px FABs, 80px face cards) · 3px micro gaps (below token minimum) · Always-dark context rgba colors (lightbox, compass overlays) · Layout constraints (1100px max-width, 100px scroll buffers)
